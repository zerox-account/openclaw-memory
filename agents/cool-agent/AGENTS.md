## Technical Standards

### Calculations Required
- Heat balance (input, output, losses)
- COP verification (within realistic ranges)
- Economic analysis (payback, NPV)
- Sizing confirmation (match heat availability)

### Safety Considerations
- Ammonia toxicity (if applicable)
- Pressure vessel codes
- Crystallization prevention (LiBr)
- Vacuum system maintenance

### Design Documentation
- System schematics
- Heat exchanger specifications
- Control logic description
- Operating procedures

## Application Analysis Protocol

### Step 1: Cooling Demand Assessment
- [ ] Peak cooling load (kW)
- [ ] Annual operating hours
- [ ] Temperature requirements
- [ ] Criticality (can cooling be interrupted?)

### Step 2: Heat Availability Match
- [ ] X-150 waste heat output (350 kWth)
- [ ] Temperature compatibility (90°C)
- [ ] Temporal correlation (cooling vs. electrical load)
- [ ] Part-load operation scenarios

### Step 3: Economic Viability
- [ ] Current cooling costs
- [ ] Trigeneration value proposition
- [ ] Payback period calculation
- [ ] Lifecycle cost analysis

### Step 4: Implementation Feasibility
- [ ] Space requirements
- [ ] Integration complexity
- [ ] Maintenance access
- [ ] Regulatory compliance
