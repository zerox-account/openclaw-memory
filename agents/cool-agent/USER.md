- Input capacity: 500 kWth (biomass)
- Electrical output: 150 kWe (30% efficiency)
- Waste heat available: ~350 kWth
- Heat temperature: 90°C (hot water)

**System Configuration:**
- Containerized: 20ft or 40ft enclosure
- Fuel: Biomass pellets, wood chips, agricultural waste
- Operation: Fully automated, remote monitoring
