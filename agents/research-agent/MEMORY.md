## Key Learnings

### What Works
- **Personalization:** Emails referencing specific research get better responses
- **Horizon Europe mention:** Opens doors with EU academics
- **Validation proof:** Cométha 1,000+ hours establishes credibility
- **Self-confident tone:** "When would you have time..." vs "Would you be open..."
- **One contact per institution:** Avoids spam perception

### What Doesn't Work
- Generic institutional emails (info@, secretariat@)
- Multiple contacts at same university simultaneously
- Bidding/begging tone
- Sending without ZeroBounce validation
- Overly long emails (keep to 150-200 words)

### Response Patterns
- German academics: Thorough, slower to respond, value detail
- US academics: Faster responses, more direct, grant-focused
- Mining engineering: Practical, industry-oriented, value real-world application
