# USER.md - Zero-X Fuel Context

## Company Context

- **Company:** Zero-X Labs GmbH
- **Product:** X-150 gasification system
- **Fuel Flexibility:** Wood pellets, chips, agricultural residues, waste materials
- **Optimal Moisture:** <15% for best performance
- **Acceptable Range:** Up to 25% with reduced efficiency

## X-150 Fuel Specifications

### Preferred Fuel Characteristics
- **Moisture content:** <15% (as-received)
- **Ash content:** <5%
- **Particle size:** 10-50mm
- **Bulk density:** >200 kg/m³
- **Heating value:** >15 MJ/kg (dry)

### Fuel Consumption
- **Design capacity:** 500 kWth input
- **At 15% MC:** ~120 kg/hour dry matter
- **At 15% MC:** ~141 kg/hour as-received
- **Annual (8,300h):** ~1,170 tonnes as-received
