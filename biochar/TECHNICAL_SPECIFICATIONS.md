**Document Version:** 1.0

## Executive Summary

This document outlines the technical specifications for biochar produced from the X-150 gasifier using digester sludge feedstock. Understanding these parameters is essential for certification, market positioning, and customer education.

**Key Characteristic:** Zero-X biochar is a high-ash, moderate-carbon biochar suitable for EBC-Basic certification and carbon credit markets.

## 2. Feedstock Specifications

### 2.1 Digester Sludge Characteristics

**Typical Composition:**

| Parameter | Range | Unit |
|-----------|-------|------|
| **Total Solids (TS)** | 15-25 | % wet weight |
| **Volatile Solids (VS)** | 60-80 | % of TS |

### 2.2 Pre-Treatment Requirements

| Step | Specification | Purpose |
|------|-------------|---------|
| **Dewatering** | TS >20% | Reduce moisture for gasification |
| **Drying** (optional) | TS >40% | Increase efficiency |
| **Size Reduction** | <50mm particle size | Ensure feeding |
| **Screening** | Remove contaminants | Protect equipment |

### 2.3 Quality Control

**Pre-Gasification Testing:**
- Moisture content (daily)
- Ash content (weekly)
- Heavy metals screening (monthly)
- Contaminant inspection (continuous)

## 4. Quality Testing Protocol

### 4.1 In-House Testing (Routine)

| Test | Frequency | Method | Equipment |
|------|-----------|--------|-----------|
| **Moisture Content** | Every batch | Gravimetric | Drying oven |
| **pH** | Every batch | Electrometric | pH meter |
| **Electrical Conductivity** | Weekly | Electrometric | EC meter |
| **Bulk Density** | Weekly | Volumetric | Graduated cylinder |
| **Visual Inspection** | Continuous | Visual | N/A |

### 4.2 Laboratory Testing (Certification)

| Test | Frequency | Method | Lab Cost |
|------|-----------|--------|----------|
| **Total Carbon** | Quarterly | Elemental analysis | €150-250 |
| **H:C Ratio** | Quarterly | Elemental analysis | Included |
| **O:C Ratio** | Quarterly | Elemental analysis | Included |
| **Ash Content** | Quarterly | Gravimetric | €50-100 |
| **Heavy Metals** | Quarterly | ICP-MS | €300-500 |
| **PAH (16 compounds)** | Quarterly | GC-MS | €400-600 |
| **Surface Area** | Annually | BET method | €200-300 |
| **Particle Size** | Quarterly | Sieving | €50-100 |

### 4.3 Sampling Protocol

**Composite Sample Collection:**

1. **Sample Points:**
   - Collect 10-20 sub-samples per batch
   - From different locations in storage
   - Equal volumes from each point

2. **Sample Size:**
   - Minimum 1 kg total composite
   - Split into three portions:
     - Analysis sample
     - Reference sample (archive)
     - Backup sample

3. **Storage:**
   - Air-tight containers
   - Cool, dry location
   - Maximum 6 months before testing

4. **Documentation:**
   - Sample ID (batch number + date)
   - Production parameters
   - Feedstock source
   - Storage conditions

## 7. Storage & Handling

### 7.1 Storage Requirements

| Parameter | Specification | Rationale |
|-----------|---------------|-----------|
| **Location** | Covered, dry area | Prevent moisture absorption |
| **Temperature** | Ambient | No special requirements |
| **Containers** | Breathable bags or bulk | Allow gas exchange |
| **Stacking** | Maximum 3m height | Prevent compaction |
| **Duration** | <12 months optimal | Prevent oxidation |

## 9. Troubleshooting Guide

### 9.1 Common Issues

| Issue | Possible Cause | Solution |
|-------|---------------|----------|
| **Low Yield** | Too high temperature | Reduce air ratio |
| **High Moisture** | Poor drying | Improve dewatering |
| **High PAH** | Incomplete combustion | Increase temperature |
| **Heavy Metals** | Contaminated feedstock | Source cleaner sludge |
| **Poor Flow** | Fine particles | Screen or granulate |
| **Odor** | Incomplete pyrolysis | Increase residence time |

### 9.2 Quality Deviations

| Deviation | Cause | Action |
|-----------|-------|--------|
| **Ash >50%** | Feedstock issue | Change source or blend |
| **Carbon <25%** | Over-gasification | Reduce intensity |
| **pH >11** | High ash minerals | Blend with acidic materials |
| **PAH >12 mg/kg** | Low temperature | Increase pyrolysis temp |
| **Heavy metals high** | Contamination | Pre-screen feedstock |
