**Author:** Clemens Grosjean
**Date:** 2026-01-15
**Status:** Published
**Featured Image:** cometha-facility.png

### Excerpt
Paris pilot demonstrates consistent oxy-steam gasification with variable feedstocks, validating X-150 system performance for commercial deployment.

### Content
The Cométha project has achieved a significant milestone, completing over 1,000 operating hours of validated oxy-steam gasification at the SIAAP facility in Paris.

**Project Overview:**
- **Location:** Paris, France
- **Funders:** Syctom, SIAAP
- **Duration:** 1,000+ operating hours
- **Process:** Oxy-steam gasification
- **Feedstock:** Variable biomass and waste streams
- **Scale:** 15 kg/h pilot system

**Validation Partners:**
- Fraunhofer Institute for Environmental, Safety, and Energy Technology (UMSICHT)
- Hochschule Zittau-Görlitz (Prof. Dr. Dr. Tobias Zschunke)

**Key Results:**
- Consistent syngas quality across variable feedstocks
- Oxy-steam process stability demonstrated
- Tar reforming efficiency validated
- Multi-fuel capability confirmed

**Significance:**
The Cométha validation provides the technical foundation for commercial deployment of X-150 systems across Europe, demonstrating reliable performance in real-world conditions.

### Content
We are pleased to release the complete technical specifications for our flagship X-150 Modular Gasification System.

**System Overview:**
The X-150 is a containerized, single-stage downdraft gasification system designed for decentralized waste-to-energy applications.

**Specifications:**

| Parameter | Value |
|-----------|-------|
| System Type | X-150 Modular Gasifier |
| Configuration | 20ft containerized |
| Gasification Process | Single-stage downdraft |
| Tar Reforming | Integrated reformer |
| Operating Mode | Air or Oxy-Steam |
| Thermal Output | ~650 kW |
| Efficiency | 86% thermal |
| Feedstock | Multi-fuel capable |
| Ash Handling | High-ash optimized |

**Key Features:**
- Plug-and-play installation
- Remote monitoring capability
- Automated feedstock handling
- Continuous operation (24/7)
- Low maintenance design

**Applications:**
- Industrial waste processing
- Agricultural residue utilization
- Sewage sludge treatment
- Biogas plant optimization
- Distributed power generation
