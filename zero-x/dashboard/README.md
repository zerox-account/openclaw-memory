# ZERO-X DASHBOARD
## Fast Money Control Center

### Features
- **FAST MONEY** Tab: Schnelle Deals (Golden Foods, DBFZ, COOLSTACK, Klärschlamm)
- **PIPELINE** Tab: Research Outreach Status (53 kontaktiert, 9 Antworten)
- **ACTIONS** Tab: Tägliche To-Do's mit Zeitangaben

### Design
- Dark Mode (Minimalistisch)
- Neon Green/Blue Akzente
- Fokus auf schnelle Übersicht
- Mobile Responsive

### Installation

```bash
cd zero-x/dashboard
npm install
npm start
```

### Oder einfach öffnen:
1. `Dashboard.jsx` und `Dashboard.css` sind die Hauptdateien
2. In React App einbinden
3. Daten anpassen in `Dashboard.jsx`

### Schnellstart

Die Daten sind direkt in der Datei editierbar:

```javascript
// Fast Money Opportunities
const fastMoneyOpps = [
  {
    name: "Golden Foods Uganda",
    value: "€2.4M",
    timeline: "30 days",
    action: "Finalize financing"
  }
];
```
