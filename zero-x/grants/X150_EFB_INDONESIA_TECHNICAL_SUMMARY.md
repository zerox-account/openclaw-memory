# TECHNICAL SUMMARY: X-150 GASIFICATION OF EMPTY FRUIT BUNCHES (EFB)
## Palm Oil Waste Valorization in Indonesia
## For Grant Applications | Prepared for Rudolph

---

## EXECUTIVE SUMMARY

The X-150 modular gasification system provides a proven, containerized solution for converting Empty Fruit Bunches (EFB) - a major waste stream from Indonesia's palm oil industry - into clean energy. With 1,000+ validated operating hours and Fraunhofer certification, the X-150 offers immediate deployment capability for waste-to-energy projects.

---

## 1. EMPTY FRUIT BUNCHES (EFB) CHARACTERISTICS

### 1.1 Biomass Profile
| Parameter | Typical Value | X-150 Compatibility |
|-----------|---------------|---------------------|
| Moisture content | 60-70% (fresh) | ✅ Integrated drying system |
| Ash content | 5-10% | ✅ Patented clinker crushing at 400°C |
| Cellulose | 35-45% | ✅ Excellent gasification substrate |
| Hemicellulose | 20-30% | ✅ High reactivity |
| Lignin | 20-25% | ✅ Good energy density |
| Calorific value (dry) | 17-19 MJ/kg | ✅ Comparable to wood pellets |

### 1.2 Availability in Indonesia
- **Annual production:** ~50 million tonnes (wet basis)
- **Geographic concentration:** Sumatra, Kalimantan, Sulawesi
- **Current disposal:** 40% burned (air pollution), 30% landfilled, 30% composted
- **Waste liability:** Palm oil mills pay $5-15/tonne disposal costs

## 2. X-150 TECHNICAL SPECIFICATIONS FOR EFB

### 2.1 System Overview
- **Type:** Containerized downdraft gasification with tar reformer
- **Footprint:** 40ft standard container (12m x 2.4m x 2.9m)
- **Capacity:** 150 kg/h dry biomass input
- **Deployment:** Plug-and-play, 2-week installation

### 2.2 EFB Processing Capability
| Processing Stage | X-150 Solution | Benefit |
|------------------|----------------|---------|
| **Drying** | Integrated oil-bath dryer | Handles 60-70% moisture |
| **Size reduction** | Shredder inlet | Accepts whole bunches |
| **Gasification** | Patented downdraft reactor | 800-1000°C operation |
| **Clinker handling** | 400°C crushing system | Handles 5-10% ash content |
| **Tar reforming** | SYNGAPURE™ catalyst | <10 mg/Nm³ tar in syngas |
| **Gas cleaning** | Pulse filter + scrubber | Meets EU emission standards |

### 2.3 Performance with EFB (Projected)
| Output Mode | EFB Input | Daily Output | Annual Output |
|-------------|-----------|--------------|---------------|
| **Electricity (VOLT)** | 150 kg/h dry EFB | 150 kW continuous | 1,314 MWh |
| **Thermal (HEAT)** | 150 kg/h dry EFB | 150 kW thermal | 1,314 MWh |
| **Hydrogen (H2)** | 150 kg/h dry EFB | 15-20 kg/day | 5.5-7.3 tonnes |
| **Biochar** | Byproduct | 30-50 kg/day | 11-18 tonnes |

**Note:** EFB requires pre-drying to ~15% moisture for optimal performance. Fresh EFB (60-70% moisture) yields approximately 40-50% of above outputs without drying, or full output with integrated drying system.
