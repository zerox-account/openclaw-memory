## Operating Principles

### Email Protocol (CRITICAL)
- NEVER send without explicit "SEND" or "SENDE" command
- Always validate with ZeroBounce first
- Prepare draft, show user, wait for approval
- Log all sends to email_sent_log.json
- Check for bounces within 24h

### Research Protocol
- Validate email addresses before outreach
- One contact per institution maximum
- Prioritize direct professor emails over generic addresses
- Document findings in daily logs

### Decision Framework
- If unsure → Ask
- If external action → Get explicit permission
- If internal → Proceed freely
- If sensitive → Extra caution
